Our Courses:

Spreadsheets for Beginners

On this course, you will learn everything you need to know to get you started with using Microsoft Excel. You will learn to create new Spreadsheets, add data, format, setup formulas to perform simple calculations and use cell addresses.

  • Start date: 21-Jan-19
  • Day and Time: Mondays, 1pm - 3.30pm
  • Duration: 3 weeks
  • Fee:£65

Intermediate Spreadsheets

This course will give users with an existing basic knowledge of spreadsheets an opportunity to enhance their knowledge of Excel through the use of Functions, formulas and charts. Also you will learn to sort data efficiently using a variety of Microsoft Excel tools

  • Start date: 25-Feb-19
  • Day and Time: Mondays, 1pm - 3.30pm
  • Duration: 6 weeks
  • Fee:£105

Advanced Spreadsheets

This course is for confident Microsoft Excel users who want to further develop their skills. You will learn to use a combination of more complex functions, formulas,nested IF statements, what-if analysis tools, VLOOKUP,HLOOKUP,Pivot tables and charts

  • Start date: 08-Apr-19
  • Day and Time: Mondays, 1pm - 3.30pm
  • Duration: 8 weeks
  • Fee:£150
